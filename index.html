Mastermind

Module: CS141 Functional Programming (Feb 2019)
This project involved writing specific functions in the Haskell to implement Donald Knuth's algorithm in the Mastermind game. This algorithm has the ability to solve any four character codes in under 5 guesses.

Scratch Clone

Module: CS141 Functional Programming (Mar 2019)
This project involved writing a program in the functional programming language Haskell which evaluates statements and instructions, depending on the given state. The program then updates and returns the state to user. Input included assignment, if statements and instructions to retrieve data from memory.
Additional functionality I implemented included while statements, repeat statements, Booleans and the factorial function.

Maze 1

Module: CS118 Programming for Computer Scientists (Oct 2019)
In this project, I had to write a program in Java to navigate a robot through a randomly generated maze. The robot needed to avoid crashing into walls and pick the correct route to move towards the target.
In the event multiple moves made the robot move towards the target, a random move was chosen.

Maze 2

Module: CS118 Programming for Computer Scientists (Dec 2019)
This task built on Maze 1 and required the robot to carry out a depth-first search in order to find the target. I used a stack to allow the robot to efficiently backtrack and find a correct route. After the depth-first search, the robot is able to use the stack to navigate a clear route to the target.
As an additional feature, I created a program which allowed the user to control the robot using a keyboard.
